## 🎯 How to Change Themes

1. **Find the Theme Button**: Look for the theme button in your sidebar (desktop) or settings area (mobile). It shows your current theme's emoji and name.

2. **Open Theme Selector**: Click the button to open the theme picker modal.

3. **Choose Your Theme**:
   - Toggle between **Light** and **Dark** modes
   - Browse visual previews of each theme
   - See color swatches and descriptions
   - Click any theme to apply it instantly

4. **Adjust Settings**:
   - **Use contextual themes**: Automatically switch themes based on which room you're in
   - **Follow system preference**: Let your device's light/dark setting control the mode

## 🏠 Contextual Theming

When enabled, UsVerse automatically applies different themes to different parts of the app:

| Room/Feature | Theme | Why |
|-------------|-------|-----|
| Dashboard, Chat, Presence | Romantic Dusk | Signature palette for core features |
| Diary, Memory Wall | Wisteria Dream | Sacred, introspective spaces |
| Countdown, Daily | Golden Hour | Warm ritual moments |
| Play, Games | Mint Garden | Light, playful atmosphere |
| Bloom, Cinema | Blush Cream | Creative and entertainment |

**To enable**: Open the theme selector and check "Use contextual themes"

## 💡 Tips

## 🎨 Design Philosophy

Each theme is carefully crafted with:
- **WCAG AA contrast ratios** for accessibility
- **Perceptual color mixing** using OKLAB color space
- **Emotional resonance** tied to each space's purpose
- **Consistent semantic colors** for success, error, warning, and info states

## ❓ Frequently Asked Questions

**Q: Will my theme reset when I refresh the page?**
A: No! Your preference is saved in your browser's local storage.

**Q: Can I use dark mode in one room and light mode in another?**
A: Not simultaneously, but contextual theming adapts the light theme per room. Dark mode applies everywhere.

**Q: What if I don't like contextual themes?**
A: Simply disable "Use contextual themes" and pick your favorite theme manually.

**Q: Are these themes accessible?**
A: Yes! All themes meet WCAG AA contrast standards (4.5:1 minimum) for readability.

**Q: Can I create my own custom theme?**
A: Not yet, but we're considering this for a future update!
